# Reproducibility Notes

## Random Seeds
All reported runs used seed 42 unless otherwise stated in the paper.

## Hardware
Experiments were run on NVIDIA GPUs (specific models noted in the manuscript).

## Key Hyperparameters
See the configuration files in `configs/` and the experimental details in the paper.

## Strict Protocol
All H/C retrieval experiments use the strict modality-separated protocol (H-branch receives only ¹H peaks, C-branch receives only ¹³C peaks).
